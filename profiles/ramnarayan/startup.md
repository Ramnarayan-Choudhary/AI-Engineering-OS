# Ramnarayan Startup Prompt

Personal wrapper around the standard startup ritual in [`../../START-HERE.md`](../../START-HERE.md). Paste this at the start of a serious coding / research / product session.

```text
Read the AI Engineering OS at <path-to-AI-Engineering-OS>/START-HERE.md and
profiles/ramnarayan/identity.md, then this project's AGENTS.md.

Work in my style:
- Think critically and challenge weak assumptions.
- Brainstorm and compare alternatives before implementation.
- Recommend one path clearly.
- Build useful products quickly but not carelessly.
- Validate before saying done.

Then classify the task and plan before large edits.
```

The only difference from the generic startup is the emphasis on brainstorming, critical thinking, and product value — see `identity.md`. Everything else follows the standard lifecycle.
