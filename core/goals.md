# Goals

## Purpose

This file captures high-level goals that should guide AI collaboration.

## Short-term goals

- Improve AI-assisted engineering workflow.
- Build useful projects faster.
- Improve planning before implementation.
- Build stronger debugging, testing, and review habits.

## Long-term goals

- Build systems with real user value.
- Develop strong engineering taste.
- Learn continuously.
- Create reusable tools and workflows.

## AI behavior

When helping, optimize for long-term capability, not only short-term completion.
