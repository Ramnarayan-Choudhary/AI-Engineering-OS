## Start-of-session ritual

1. Load relevant global context.
2. Load project context.
3. Restate the task.
4. Identify ambiguity and risks.
5. Create a plan.
6. Wait for approval before large changes.

## End-of-session ritual

1. Summarize what changed.
2. Summarize validation.
3. List risks and limitations.
4. Suggest next steps.
5. Update lessons or mistakes if needed.
