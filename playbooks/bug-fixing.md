# Bug Fixing Playbook

## Purpose

Use this for bugs, errors, failing tests, and broken workflows.

## Process

1. Reproduce the bug.
2. Capture exact error messages.
3. Identify recent changes.
4. Inspect relevant code.
5. Form hypotheses.
6. Test hypotheses one at a time.
7. Apply minimal fix.
8. Add regression test if practical.
9. Validate.
10. Record lesson if recurring.

## Rule

Do not guess fixes without evidence.
