# Parallel Agents Playbook

## Purpose

Use this when multiple agents or worktrees explore different solutions.

## Process

1. Define shared goal.
2. Create separate branches or worktrees.
3. Give each agent a bounded task.
4. Prevent agents from editing the same files when possible.
5. Require each agent to produce a summary and validation report.
6. Compare outputs.
7. Merge the best parts manually.

## Safety rule

Parallel agents increase speed but also increase merge risk. Keep tasks narrow.
