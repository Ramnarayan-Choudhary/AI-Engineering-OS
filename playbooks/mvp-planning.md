# MVP Planning Playbook

## Purpose

Use this when turning an idea into a small product.

## Process

1. Choose one user segment.
2. Choose one painful problem.
3. Describe current alternatives.
4. Define smallest useful solution.
5. Cut non-essential features.
6. Define success metric.
7. Build prototype.
8. Get feedback.
9. Iterate.

## Rule

If the MVP cannot be explained in one sentence, it is too complex.
